# exp_001_router_eval

## Goal

Compare three documentation architectures for agent work:

- baseline
- wiki-heavy
- hybrid-router

## Prompt Suite

1. Onboard a new agent into the repo
2. Recall a prior decision
3. Explain a durable architecture concept
4. Update knowledge from a new raw source note
5. Plan around branch and experiment status

## Current Read

- Hybrid-router is the only mode that treats operational docs and durable knowledge as separate lanes
- Wiki-heavy may eventually outperform it, but not until the wiki maintenance loop is extremely mature
- Baseline is still the easiest onboarding path
